# K3s Setup

Install K3s inside the Ubuntu VM:

```bash
./scripts/k3s/20-install-k3s.sh
```

The script installs a single-node cluster and writes kubeconfig to:

```text
~/.kube/config
```

Check the cluster:

```bash
kubectl get nodes -o wide
kubectl get pods -A
```

## Remote Admin Access

The simplest path is to SSH into the Ubuntu VM and run `kubectl` there.

For MacBook/WSL access, use Tailscale first, then copy kubeconfig from the VM:

```bash
scp USER@TAILSCALE_VM_NAME:~/.kube/config ~/.kube/macmini-lab.yaml
export KUBECONFIG=~/.kube/macmini-lab.yaml
kubectl get nodes
```

If the kubeconfig contains `127.0.0.1`, replace it with the VM's Tailscale IP or
MagicDNS name.

## Firewall Notes

Do not expose the Kubernetes API publicly. Keep port `6443` private to the VM
and your Tailnet. Do not expose Flannel VXLAN traffic to the internet.
